# 工厂产量系统 - 本地部署完整教程

## 一、你需要准备什么？

### ✅ 你已经有的：
- 这台电脑（Windows 系统）
- 工厂产量系统的 `factory.html` 文件

### ✅ 需要下载的（全部免费）：
1. **Python**（服务器环境）- 后面会教你下载
2. 手机和电脑连接同一个 **Wi-Fi** 网络即可

---

## 二、第一步：下载并安装 Python

### 1. 下载 Python
- 打开浏览器访问：https://www.python.org/downloads/
- 点击黄色大按钮 **Download Python 3.x.x**（x.x.x 会变，比如 3.12.0）
- 下载完成后双击安装包

### 2. 安装时非常重要！
- ✅ **勾选第一项**：`Add Python to PATH`（这是默认不勾选的，必须勾！）
- 点击 `Install Now` 或 `Install` 按钮
- 等待安装完成（约 2-5 分钟）

### 3. 验证安装成功
- 按 `Win + R` 键，输入 `cmd`，回车
- 输入以下命令（按回车）：
  ```
  python --version
  ```
- 如果显示类似 `Python 3.12.0`，说明安装成功！

---

## 三、第二步：启动本地服务器

### 方法一：用我给你的启动脚本（最简单）

我会在你项目目录下创建一个 `.bat` 脚本，你双击就能运行：

```
start-server.bat
```

双击后会打开一个黑色窗口，显示服务器地址。

### 方法二：手动启动（备用方案）

1. **找到 factory.html 的文件夹**
   - 比如你的文件在：`c:\Users\J.s\WorkBuddy\20260323021915\factory.html`

2. **打开该文件夹**
   - 在文件夹空白处按住 `Shift` 键，右键点击
   - 选择"在此处打开 PowerShell 窗口"或"在此处打开命令窗口"

3. **启动服务器**
   输入以下命令（复制粘贴即可）：
   ```
   python -m http.server 8080
   ```

4. **看到这个就成功了**
   ```
   Serving HTTP on 0.0.0.0 port 8080 ...
   ```

---

## 四、第三步：电脑上测试

1. **在电脑浏览器打开**
   ```
   http://localhost:8080/factory.html
   ```

2. **能打开系统，说明服务器跑起来了！**

3. **不要关闭黑色窗口**，关了服务器就停了

---

## 五、第四步：让手机也能访问

### 1. 查看电脑的 IP 地址
- 在黑色窗口（服务器窗口）输入（或另开一个 cmd）：
  ```
  ipconfig
  ```
- 找到"无线局域网适配器 Wi-Fi"下面的"IPv4 地址"
- 格式类似：`192.168.1.100` 或 `192.168.0.105`

### 2. 手机连接同一个 Wi-Fi
- 确保手机和电脑连的是同一个 Wi-Fi 网络名

### 3. 在手机浏览器输入
```
http://你的IP地址:8080/factory.html
```

**举例**（如果你的 IP 是 192.168.1.100）：
```
http://192.168.1.100:8080/factory.html
```

### 4. 收藏到手机浏览器
- 手机上能打开后，收藏到书签或添加到主屏幕
- 以后直接点就能打开

---

## 六、关闭服务器

- 在黑色窗口按 `Ctrl + C` 停止服务器
- 或直接关闭黑色窗口

## 七、常见问题

### ❓ Q1：手机访问显示"无法连接"
**A**：检查以下几点
- 电脑和手机是不是同一个 Wi-Fi？
- IP 地址是不是输对了？
- 电脑防火墙是不是拦截了？（后面有解决方法）

### ❓ Q2：电脑上能用，手机上不行
**A**：这是防火墙问题
- 打开"Windows Defender 防火墙"
- 左侧点击"允许应用或功能通过 Windows Defender 防火墙"
- 点击"更改设置"
- 找到"Python"（两个勾都打上）
- 如果没有 Python，点击"允许其他应用"，找到 `python.exe`

### ❓ Q3：启动服务器时提示端口被占用
**A**：端口 8080 被其他软件占用了
- 换个端口试试，比如：
  ```
  python -m http.server 8888
  ```
- 然后手机访问变成：
  ```
  http://192.168.1.100:8888/factory.html
  ```

### ❓ Q4：我换了 Wi-Fi，手机上打不开了
**A**：IP 地址变了
- 重新查 IP（看第四步）
- 更新手机上的收藏地址

### ❓ Q5：每次开机都要手动启动？
**A**：可以开机自启动
- 把 `start-server.bat` 复制到：
  ```
  C:\Users\你的用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
  ```
- 但要接受开机后会有一个黑色窗口（关了服务器就停了）
- **更好的方案**：把电脑设置为"永不休眠"，服务器一直挂着

## 八、进阶：让外网也能访问（比如在家里用手机访问公司的电脑）

这个比较复杂，需要：
- 路由器有公网 IP
- 配置端口映射
- 或使用内网穿透工具（如 frp、ngrok）

**对于新手不推荐**，局域网够用了。

## 九、快速启动脚本（推荐使用）

我会给你创建一个 `.bat` 文件，内容如下：

```batch
@echo off
chcp 65001 >nul
echo ====================================
echo    工厂产量系统 - 本地服务器
echo ====================================
echo.
echo 正在启动服务器...
echo.
echo 电脑访问地址：http://localhost:8080/factory.html
echo.
echo 手机访问地址（手机和电脑需连同一 Wi-Fi）：
echo 先查询电脑 IP，然后访问：http://IP地址:8080/factory.html
echo.
echo 按 Ctrl+C 可停止服务器
echo ====================================
echo.

python -m http.server 8080

pause
```

保存为 `start-server.bat`，双击即可运行。

## 十、总结

**启动流程：**
1. 确保电脑和手机连同一个 Wi-Fi
2. 双击 `start-server.bat` 启动服务器
3. 电脑上访问：`http://localhost:8080/factory.html`
4. 查询电脑 IP
5. 手机上访问：`http://IP地址:8080/factory.html`

**就这么简单！** 不需要买服务器，不用花钱。

## 十一、数据保存在哪里？

你的所有数据都保存在浏览器的 **LocalStorage** 里：
- 电脑的数据在电脑的浏览器里
- 手机的数据在手机的浏览器里
- **它们是分开的，不同步**

**如果你想在手机上操作后，电脑上也看到数据：**
- 要么只用一台设备登录
- 要么用管理员导出 Excel，再导入（目前系统支持导出 Excel）
